# 概述

Neomark 是一种轻量级的标记语言，旨在保证纯文本可读的前提下，提供丰富的富文本标记与足够的拓展性。

本手册主要使用自然语言对文档的解析进行描述，这是旨在便于理解语法本身，这意味着本文档可能导致一些歧义。用[《Python 语言参考手册》](https://docs.python.org/zh-cn/3.14/reference/introduction.html)的话讲：

> 如果你是来自火星并且想凭借这份文档把 Neomark 重新实现一遍，也许有时需要自行猜测，实际上最终大概会得到一个十分不同的语言。

但本文档始终是 Neomark 解析器实现的最终标准，对于一些细节的行为，可以参考本项目对 Neomark 解析器的实现。也因此，本手册也偶尔提及本项目实现的细节，以表明 Neomark 解析器在处理边界情况时的行为。
