# 文件单位

Neomark 的文档单位是指能被 Neomark 解析器利用的一系列文件，主要包括：

1. **文档**：符合本手册的 Neomark 文档。
2. **程序**：使用脚本语言编写，符合本手册API规范且可被调用的脚本。
3. **资源**：可被文档引用、链接或嵌入展示的外部数据文件。

本手册不限制文件单位究竟以何种方式存储，这些文件至少被组织在目录结构中，且记录了以下元数据：

1. 文件名
2. 创建时间
3. 修改时间

对于存储在文件系统中的文件单位，Neomark推荐的后缀名如下：

| 类型 | 后缀 |
| --- | --- |
| 文档 | `.nmd` |
| 程序 | `.nms` |
| 资源 | 该资源对应的正确后缀 |
